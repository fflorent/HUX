Available modules

  • The Core : the common functions
  • Form : a script for validating forms using Ajax
  • Hash Manager : History Manager using the Hash of the URL
  • Overlay : a module to update any element of a web page chosen server-side
  • Script Injecter : a module to inteprete javascript files when added with HUX
  • Simple Loader : a module to load content (no history management)
  • Stage Class Manager : a module that adds classes to a target at each stage of a load using HUX
  • Transition : a module to make transitions while contents are loading
  • URL Manager : History Manager using the new History API
  • URL Manager Fallback : a module that converts links for URL Manager to links for Hash Manager
