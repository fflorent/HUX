Url Manager

Warning : Url Manager requires Firefox 4+, Chrome 5+, Opera 11.50+, and Safari5+

Url Manager is a plugin for HUX that updates the URL at the same time that you load contents.

How To Use it

First, you need to create an .htaccess file (or an equivalent) with RewriteRules. The minimum .htaccess file that you must have looks like below :

RewriteEngine on
RewriteRule ([^@]*)@.* $1
#other RewriteRules ...

Then, you can simply use it by creating links of this syntax :

<a href="@TARGET_ID=URL_OF_YOUR_CONTENT">TEXT</a>

The syntax and the behaviour are similar to the Hash Manager ones.

Example

Example 1

Code :

<a href="@ex_um_target1=res_hello_world">click to run Example 1</a>
<div id="ex_um_target1">I will disapear...</div>

Rendering :

click to run Example 1
I will disapear...

Known Bugs

When the URL of the content to load contains slashes, some unexpected behaviour appears once the content is loaded (cannot reload anymore, links using URL Manager do not work anymore, etc...).
To avoid this problem, use your .htaccess file to write rules so the URL does not contain slashes.
For example, if you want @feature=features/any.html, you can transform it as @feature=f_any, and edit .htaccess file as below:

RewriteEngine on
RewriteRule ([^@]*)@.* $1
RewriteRule ^f_(.*) features/$1.html
