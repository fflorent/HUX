As a first step

Simple HUX Request

SimpleLoad is a module that lets you do simple requests and update your page with the loaded content.

As a first HUX request, we will do this :

Say Hello!

...

Lets take a look at the code :

1. <p><a href="resources/hello_world.html" data-hux-target="HUXdiv1">Say Hello!</a></p>
2. <div id="HUXdiv1">...</div>

We have got :

  1. a link <a> with an href attribute and another attribute data-hux-target, which is specifique to HUX.
  2. the target element, where the content will be injected.

The data-hux-target attribute indicates the target where HUX must inject the content.
In our case, we say to HUX "when user click on the link, inject the content of resources/hello_world.html in the element whose ID is HUXdiv1.

NOTE : You can also use filling methods with SimpleLoad
