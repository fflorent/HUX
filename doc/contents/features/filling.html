Filling

data-hux-filling is an attribute that specifies the filling method of a HUX request.

By default, HUX supports these three filling methods:

  • replace : the new content replaces the old one
  • prepend : the new content is injected after the old one, and the old one remains
  • append : the new content is injected before the old one, and the old one remains

Feature description

Type Attribute
Attribute of A, FORM elements
Possible values "append", "prepend", "replace"
