Target

data-hux-target is an attribute that specifies the ID of the target element of the HUX request.

This target element will be injected the loaded content according to the filling method.

Feature description

Type Attribute
Attribute of A, FORM elements
Possible values The ID of any element in the BODY of the HTML Document

Examples

Code :

<a href="#!ex_target_target1=resources/hello_world.html">click to run Example</a>
<div id="ex_target_target1">I will disapear...</div>

Rendering :

click to run Example
I will disapear...
