The Form Module

When the user submit a form in your website, the Form Module lets you update your page with the content returned by the server

The most eloquent example is when submitting a comment to an article. In most cases, client side, you just want to append the submitted comment in the comments list, and not reload the entire webpage.
That is one of the use case the Form Module provide a solution for.

How to use it

Its use is quite similar to Simple Load.
You have to create a form as you usually do, and then add data-hux-target and data-hux-filling attributes to the <FORM> element.
So it has this form :

<form method="[GET|POST]" action="URL" data-hux-target="target_id"
data-hux-filling="[replace|prepend|append]">
<!-- INPUTS HERE -->
</form>

Where :

  • URL is the URL you submit data to. NOTE : cross-domain requests are not supported yet;
  • target_id is the id of the target element, which will receive the content of what returns the server:

The filling method can be :

  • "replace" to replace the content of the target element
  • "append" to append the loaded content in the target element
  • "prepend" to prepend the loaded content in the target element
