The demos presented here use HUX and are intended to demonstrate its potential, a framework to Ajaxise a website.

Demos are accessible via the left menu.

Since HUX is still in an experimental state, you can encounter some bugs. Note that these demos have been tested mostly on Mozilla Firefox 6.

If you are interested in HUX and you are a Javascript developer, you are welcome for helping me to develop it.
