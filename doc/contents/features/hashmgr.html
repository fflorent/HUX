Hash Manager

The Hash Manager is a plugin for HUX that do HUX requests when you update the Hash of your URL.

This plugin is recommended when you need to store in your URL the contents that are injected via HUX in your document.

NOTE : This plugin can only replace the content of a target element (no append nor prepend of content).

How To Use it

If you enabled it in the download page, you can simply use it by creating links of this syntax :

<a href="#!ID_TARGET=URL_OF_YOUR_CONTENT">TEXT</a>

Where :

  • ID_TARGET is the ID of your target element (see target to understand what it is);
  • URL_OF_YOUR_CONTENT is the URL of the content you want to load;

WARNING : HUX does not support cross-domain HTTP Requests. This means that you can only load content of your own site.

NOTE : if your old URL already contained !ID_TARGET=... , the last will be removed and replaced by #!ID_TARGET=URL_OF_YOUR_CONTENT

You can also accumulate different ID of targets in the hash or in the href attribute of a link.

To sum-up, the URL will have this form :

http://your_domain.com[:port]/your/pathname/page.html#!target1=url1,!target2=url2,...
