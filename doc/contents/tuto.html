Before beginning to learn how to use HUX, you need to have a basic knowledge of HTML.
If you do, that should be fine =)

Simple HUX Request

As a first HUX request, you will do this :

  • Say Hello!
  • ...

To do that, you will need to make :

  • a link with a target attribute.
  • a target element with an id

A target element is the element that will receive the HTML content requested. The attribute target contains the ID of the target element.

In other word, if you have the following target element :

<div id="myTN"><div>

you would need a link of this type :

<a href="/to/my/content.html" data-hux-target="myTN">click me<a>


So, by clicking on the link, the user will load content.html in the div element (named target node because it is the node that will receive the content)
