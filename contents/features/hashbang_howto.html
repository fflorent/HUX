Hash Bang : How to use it ?

The use of Hash Bang is very similar to At Inclusion, except that :

  1. links do not begin with @ but with # (hash)
  2. each target must be preceded by a ! (bang)

For example :

<a href="@target1=url1,target2=url2">link text</a>

has for Hash Bang equivalent :

<a href="#!target1=url1,!target2=url2">link text</a>

Setting Hash Bangs

Example :

<a href="#!target1=url1,!target2=url2">link text</a>

Using operations

To add a new Hash Bang pair, use a plus (+) after the bang (!):

<a href="#!+target1=url1">link text</a>

To remove a pair (and reset the default content for the target), use a minus (-) after the bang (!) :

<a href="#!-target1">link text</a>

For a detailed How To, look at the At Inclusion documentation, "Operations with At Inclusions" section
