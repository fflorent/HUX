Simple Load

Simple load is a light module to create links that loads remote content. So there is no URL update, no history management.

How to use it

The link using Simple Load must have this form at a minimum :

<a href="url" data-hux-target="target_id">Link text</a>

Where :

  • target_id is the ID of the target element (i.e. the one which will receive the content)
  • url is the URL of the content to load

By default, Simple Load replaces the content of the target element. However, you can also specify the filling method with data-hux-filling attribute :

<a href="url" data-hux-target="target_id" data-hux-filling="[replace|append|prepend]">Link text</a>

The possible values for data-hux-filling are :

  • "replace" to replace the content of the target element
  • "append" to append the loaded content in the target element
  • "prepend" to prepend the loaded content in the target element
