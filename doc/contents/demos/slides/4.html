History Management

  • History Management lets the visitor use the Previous/Forward button of their browser and save in the URL the visitor has requested (so he can refresh or share the URL)

  • HUX has currently two modules to manage the History : Hash Manager and URL Manager
    • Hash Manager is compatible with almost all the browsers. It updates the Hash of the URL to save
    • URL Manager use the new History API (history.pushState, etc.), so it is currently only compatible with Firefox 4+, Chrome 5+, Opera 11.50+, and Safari5+
